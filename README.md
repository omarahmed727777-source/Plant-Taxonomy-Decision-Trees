# Plant Taxonomy Prediction via Decision Trees
Using Decision Tree classifiers to categorize botanical data based on physical attributes.

## Key Features
- **Tree Visualization:** Visualized the decision logic and tree structure for better interpretability.
- **Model Metrics:** Attained **100% Accuracy and F1-Score** on the Iris dataset.
- **Techniques:** Applied tree depth control to manage model complexity.

## Technologies Used
- Scikit-learn (DecisionTreeClassifier), Matplotlib.
